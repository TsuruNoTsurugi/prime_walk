# prime_walk
Programs to generate Prime Walk Arts

## Usage
```bash
$ python3 prime_walk.py
```
```bash
$ python3 prime_ulam.py
```

## Files
This program requeres the prime number list file `prime_num.txt` to be in the same directory, and splitted by comma.
